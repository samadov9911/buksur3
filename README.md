# 🚀 ORBITAL TUG — Симулятор космического буксира

Реалистичная развлекательная и познавательная игра о космических буксирах.
Доставка наноспутников на орбиту и уборка космического мусора.

## 🎮 Режимы игры

| Режим | Описание |
|-------|----------|
| 🛰 **NANOSAT DEPLOYER** | Доставка наноспутников CubeSat (1U/2U/3U) на заданные орбиты с реалистичной орбитальной механикой |
| 🗑 **SPACE JANITOR** | Захват и утилизация реального космического мусора (24 объекта) гарпуном, манипулятором или сетью |

## 🛠 Технологии

- **Next.js 16** (App Router, TypeScript)
- **React Three Fiber** — 3D-визуализация орбит, Земли, спутников
- **Tailwind CSS 4** + shadcn/ui
- **Zustand** — управление состоянием игры
- **Prisma** (SQLite) — лидерборд
- **Framer Motion** — анимации UI

## 📦 Установка и запуск

```bash
# Установить зависимости
npm install

# Инициализировать базу данных
npx prisma db push

# Запустить dev-сервер
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000)

## 🚀 Деплой на Vercel

1. Создайте репозиторий на GitHub
2. Импортируйте его в [Vercel](https://vercel.com)
3. Установите переменные окружения (если нужно):
   - `DATABASE_URL` — для лидерборда (опционально)
4. Deploy!

## 🎯 Управление

### Десктоп
| Клавиша | Действие |
|---------|----------|
| WASD | Ориентация (тангаж / рысканье) |
| A + D | Крен |
| ↑↓ | Тяга вперёд/назад |
| Q / E | Тяга вверх/вниз |
| Пробел | Захват / Выход из режима |
| V | Переключение камеры |
| + / - | Изменение высоты орбиты |
| [ / ] | Ускорение времени |
| ESC | Пауза |

### Мобильные устройства
- Виртуальный джойстик для ориентации и тяги
- Кнопки действий на экране
